## exp3.y
```
%{
#include <stdio.h>

// Declare yylex() and yyerror() here to avoid warnings
int yylex(void);
void yyerror(const char *s);

%}

%token ID PLUS MINUS MULTIPLICATION DIVISION

%%

statement: ID '=' E {
    printf("\nValid arithmetic expression");
    $$ = $3;
}
;

E: E PLUS ID  { $$ = $1 + $3; }
 | E MINUS ID { $$ = $1 - $3; }
 | E MULTIPLICATION ID { $$ = $1 * $3; }
 | E DIVISION ID { $$ = $1 / $3; }
 | ID
 ;
```
